# CampusChrono - System Requirements

## Server Requirements
- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache Web Server (via XAMPP)
- PHPMailer 6.9.1 (included in vendor folder)

## PHP Extensions Required
- mysqli (MySQL database connection)
- session (Session management)
- json (JSON encoding/decoding)
- mbstring (Multi-byte string handling)
- openssl (Secure connections for SMTP)

## Software Requirements
- XAMPP (Apache + MySQL + PHP)
  Download: https://www.apachefriends.org/

## Email Requirements
- Gmail account with 2-Step Verification enabled
- Gmail App Password for SMTP authentication
  Generate at: https://myaccount.google.com/apppasswords

## Browser Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- JavaScript enabled
- Cookies enabled

## Database
- MySQL Database: notice_sender
- Import file: database/schema.sql
- Default credentials: root / (no password)

## File Upload Requirements
- PHP upload_max_filesize: 10M or higher
- PHP post_max_size: 10M or higher
- Allowed file types: PDF, JPG, PNG
- Max file size per upload: 5MB

## Folder Permissions
- uploads/notices/ - Write permission required
- config/ - Read permission required

## Network Requirements
- Internet connection (for sending emails via Gmail SMTP)
- Port 587 open (for SMTP STARTTLS)

## Optional
- SSL Certificate (for production deployment)
- Custom domain name

## Included Dependencies
- PHPMailer 6.9.1 (vendor/PHPMailer-6.9.1/)
  - PHPMailer.php
  - SMTP.php
  - Exception.php

## Installation Steps
1. Install XAMPP
2. Copy project to htdocs folder
3. Create config/config.php from config/config.example.php
4. Setup Gmail App Password
5. Import database/schema.sql
6. Access via http://localhost/NOTICE_SCHEDULER/
