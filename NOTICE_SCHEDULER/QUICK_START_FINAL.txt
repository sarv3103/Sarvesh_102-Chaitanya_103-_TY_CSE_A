EMAIL NOTIFICATIONS (4 TYPES)
===============================================

1. OTP Verification Email
   - Sent immediately after registration
   - Contains 6-digit OTP
   - Valid for 5 minutes

2. Application Submitted Email
   - Sent after OTP verification
   - Confirms application sent to admin
   - Explains next steps

3. Approval Email (Congratulations!)
   - Sent when admin approves
   - Subject: üéâ Congratulations! Your Account Has Been Approved
   - Contains login URL
   - User can now login

4. Rejection Email (Sorry)
   - Sent when admin rejects
   - Subject: Application Status: Rejected
   - Contains rejection reason
   - User account deleted
