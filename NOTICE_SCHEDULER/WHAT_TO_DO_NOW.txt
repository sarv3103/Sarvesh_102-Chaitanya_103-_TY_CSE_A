STEP 6: ENABLE EMAIL (OPTIONAL)
===============================================

If you want real emails instead of OTP on screen:

1. Open: C:\xampp\htdocs\NOTICE_SCHEDULER\config\config.php

2. Change:
   define('EMAIL_ENABLED', false);
   TO:
   define('EMAIL_ENABLED', true);

3. Configure Gmail App Password:
   - Go to: https://myaccount.google.com/apppasswords
   - Generate 16-character app password
   - Update in config.php:
     define('SMTP_USER', 'your-email@gmail.com');
     define('SMTP_PASS', 'your-16-char-app-password');
     define('SMTP_FROM', 'your-email@gmail.com');

4. Test registration again - OTP will be sent to email

===============================================
TROUBLESHOOTING
===============================================

Problem: Department dropdown empty
Solution: Run http://localhost/NOTICE_SCHEDULER/test-departments.php
          If no departments, import database/schema.sql

Problem: Registration error
Solution: Run http://localhost/NOTICE_SCHEDULER/debug-registration.php
          Check error messages

Problem: Cannot login
Solution: Run http://localhost/NOTICE_SCHEDULER/diagnose-login.php
          Check if user is verified and approved

Problem: Email not sending
Solution: Check config.php EMAIL_ENABLED setting
          If true, verify SMTP settings
